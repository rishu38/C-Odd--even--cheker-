# Odd Even Checker in C

This is a simple C program that checks whether a number is odd or even.

## Features
- Takes input from user
- Checks number using if-else
- Displays whether the number is odd or even

## Technologies Used
- C Programming Language

## How to Run
1. Open the file in CodeBlocks / VS Code
2. Compile the program
3. Run the program
4. Enter a number
